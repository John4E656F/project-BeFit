# 🏋️ project-BeFit

**Full‑stack fitness platform.** Expo TypeScript frontend + Go backend.

```
project-BeFit/
├── frontend/        # Expo TypeScript (React Native) mobile app
│   └── ...
├── backend/         # Go API server
│   ├── main.go      # Entry point
│   ├── go.mod       # Module: github.com/John4E656F/project-BeFit/backend
│   └── ...
├── README.md        # This file
└── .gitignore       # Shared ignore rules
```

---

## 🚀 Getting Started

### Backend

```bash
cd backend
go run main.go
```

### Frontend

```bash
cd frontend
npm install
npx expo start
```
